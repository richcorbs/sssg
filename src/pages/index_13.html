Welcome to

simple static site generator!

Congratulations! You've just created a web site and this is the index page.

Tips for getting started:

  • Edit the src/pages/index.html file to change this page's content. The page should hot reload automatically after you save.
  • Edit the src/layouts/Default.html file to change the default layout/wrapper.
  • Edit the src/assets/css/custom.css file to change the custom styles. We've made it easy for you to change the brand and accent colors and to import your favorite web fonts.
  • Add a page in src/pages with just markdown. Make sure it has an extension of .md.
